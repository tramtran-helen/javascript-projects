# ⚡ JavaScript Mini Projects

A curated set of fun, mini JavaScript projects that showcase core web functionality, APIs, and interactive UI behavior.

## 🚀 Projects Included

- 🃏 **Blackjack Game** – A browser-based blackjack experience with basic game logic.
- 📝 **Blog Space** – Create and update blog entries with a clean, minimal interface.
- 🎲 **Bored Bot** – Generates random activities using the Bored API — perfect for quick inspiration!
- 💼 **Business Card** – A digital personal card layout with structured content and styling.
- 🔗 **Chrome Extension** – A lead tracker tool to save and manage useful links from your browser.
- 👥 **Passenger Counter** – A click-based counter for tracking people or events in real time.
- 📱 **Track Mobile App** – A mobile-style UI demo for tracking features or content.
- ⚔️ **War Game** – A two-player card game powered by the Playing Card API.
